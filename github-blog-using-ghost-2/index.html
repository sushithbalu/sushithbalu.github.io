GitHub Pages Blog using Ghost

28 September 2016

There are plenty of ways to make blogs or websites. But, building a blog on GitHub pages using ghost blogging platform is the most elegant approach to do it.

  • Generate appropriate directories, HTML pages and links for the blog using buster tool.
The 'Ghost'

Ghost is an open source blog publishing platform with lot of awesome features and themes. It is very easy to write and publish content on it.

You can be downloaded and extracted ghost using following commands in a Linux machine.

wget https://ghost.org/zip/ghost-0.11.0.zip
unzip ghost-0.11.0.zip -d ghost
cd ghost

Or you can download a development copy of ghost from here. Now, save and unzip the file to a safe location.
Follow these links to install Node.js and npm.

Ghost is written in javascript language. So we need Node.js and npm to install and run Ghost. Node.js is the runtime environment for javascript components and npm is the package manager which is used to install, publish and manage Node.js packages.

If you are using windows or mac npm can be installed directly using pre-built npm installers.

Now, you can install and start ghost in local machine using following commands. Default ULR for Launching local site from browser will be http://localhost:2368. 

npm install --production
npm start

Then, Set-up user Credentials, theme. You can writing posts using markdown editor. This editor is simple and lot of short cut keys are available.

GitHub Pages

GitHub pages free platform which helps to build websites or blogs for your projects. However you can create only one website per GitHub account.

First, head over to GitHub and create a new repository named username.github.io, where username is your username (or organization name) on GitHub. The first part of repository should match with your user name. Otherwise, it won't work.

For adding blog to GitHub pages, Clone the repository that you have created to your local machine. Using buster generate static folder, deploy this inside the repository, commit and push changes to GitHub. Done!. After some time you will be able to launch your website at http://username.github.io.

DISQUS: a comments plugin

We cannot directly add comment box in GitHub Pages blog or website. Because, these are static pages. However we can use third party service like DISQUS to do it.

To add this plugin you need to register with DISCUS fisrt. Once you registered successfully, they will provide a code snippet to insert in your blog. This DISQUS help page will help you to do that.

Now, locate the file post.hbs inside theme directory in locally downloaded Ghost folder. (Path will be like /content/themes/[theme_name]/post.hbs) and paste the code snippet under {{/post}} and < /article>. Start the local site, generate static pages using buster and commit the code to GitHub Pages. That's it!, Now you will be able to see comment box under your blog posts.
